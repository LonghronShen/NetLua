Have you ever wanted to use Lua in your .NET projects but didn't want to use native code wrappers?

NetLua is a free implementation of the Lua dynamic language in C#

Getting started

{% highlight c# linenos %} var lua = new NetLua.Lua(); lua.DynamicContext.var = LuaObject.NewTable(); lua.DynamicContext.var.x = 4; lua.DoString("var.x = 4"); Console.WriteLine(lua.DynamicContext.var.x); {% endhighlight %}
